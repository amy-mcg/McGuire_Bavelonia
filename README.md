# McGuire_Bavelonia
Data files and associated code developed for McGuire et al. (in prep), an output of the ERC-funded RISeR project at the University of Leeds (https://riser.leeds.ac.uk). Details of the RISeR cores, including core imaging and laminography can be retrieved from Zenodo (https://zenodo.org/records/15584041).

## Palaeomagnetism
Contains the palaeomagnetic data from RISeR_S7 and RISeR_S6 formatted for [Palaeomagnetism.org](https://paleomagnetism.org).

## Pollen
Contains the pollen data from RISeR_S1, RISeR_S7, RISeR_S6, and RISeR S4. Pollen data for RISeR_S1, RISeR_S7, and RISeR_S6 were generated by Amy McGuire at the School of Earth and Environment at the University of Leeds (Leeds, UK). Pollen data for RISeR_S4 were generated by Frans Bunnik at TNO-Geological Survey of the Netherlands (Utrecht, The Netherlands).
